---
title: Focus mode — silencing notifications during Rehearsal
summary: A Rehearsal run puts your phone into Do Not Disturb for the run, then restores it automatically.
date: 2026-07-21
---

Starting a Rehearsal run turns on Do Not Disturb so notifications don't interrupt you
mid-scene — alarms and priority contacts still get through. It's Rehearsal-only; Practice and
Teach never touch your phone's notification settings.

## Granting access

The first time, a "Focus mode" dialog asks to allow Actors Lines to silence notifications while
you rehearse. "Allow" opens the system's notification access settings so you can turn it on;
"Not now" skips it for this run and you'll be asked again next time. Without it granted,
Rehearsal runs exactly as normal — you just won't get the notification silencing.

## Restoring your notifications

- Whatever filter you had running before is restored the moment the run ends — leaving the
  screen, backgrounding, or the run being stopped or completing all count.

> **Tip:** This can't get stuck on: even if the app is killed outright mid-run — a crash, a
> force-stop, swiping it away from Recents, or the system killing it for memory — the next time
> you open the app it checks and restores your previous filter if Rehearsal never got the chance
> to.
