Literata is the default reading font, but you can swap it for anything from a
screenplay-style typewriter face to a plain, highly legible sans — whatever reads best for you.

## Changing it

- Open Settings → Appearance and tap the "Font" row.
- A live preview line sits at the top of the screen; tap any font in the list below and it
  updates immediately — there's no separate save step.

## The choices

- Literata (default), EB Garamond, Source Serif 4, Lora, PT Serif — serif fonts, from classic
  book type to a familiar office-document look.
- Courier Prime — the typewriter monospace used in the film/TV screenplay industry.
- Inter, Open Sans, Source Sans 3, Atkinson Hyperlegible — sans-serif fonts, the last one
  designed for maximum character legibility.

> **Tip:** This only changes the script text itself — dialogue, character cues and stage
> directions. The app's own buttons and menus stay as they are.
